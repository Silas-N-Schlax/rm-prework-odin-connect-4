# Running the Project Locally

Clone the repository:

```bash
git clone https://github.com/Silas-N-Schlax/rm-prework-odin-connect-4.git
```

Navigate to the project directory:

```bash
cd rm-prework-odin-connect-4
```

Install RSpec, Colorize, and Rubocop
```bash
bundle install
```

Run `ruby main.rb` in your terminal.

Run `rspec spec/connect_four_spec.rb` in terminal for main spec file.
